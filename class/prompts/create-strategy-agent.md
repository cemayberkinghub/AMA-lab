# Prompt: Create Strategy Agent

Use this prompt during workbook section 5, step B. Copy it into GitHub Copilot CLI to draft `strategy.agent.md`.

## Copy/paste prompt

```text
# Lab Task: Create Strategy Agent
## Goal
Turn your inputs into a strategy-focused agent for GitHub Copilot CLI.

## Output
Write as a GitHub Copilot custom agent file, using YAML frontmatter and Markdown instructions, ready to copy and paste to:

$HOME\.copilot\agents\strategy.agent.md

## Structure requirement
- Follow GitHub Copilot custom agent structure documented by GitHub:
  https://docs.github.com/en/copilot/concepts/agents/coding-agent/about-custom-agents

## Include, where applicable
- Strategic reasoning style
- Decision framing
- Tradeoff posture
- Communication style
- Strong opinions and core principles
- Red lines
- When to escalate

## Exclude
- Anything already covered in shared `copilot-instructions.md`
- CSA-specific technical architecture review behavior

## Constraints
- Strategy role only
- No duplication of shared guidance
- Keep it concise, practical, and actionable
```
